# CardGame

Basic 5 card stud poker game using HTML, CSS and JQuery

## Installation

None - just open or drop the index.html file into browser url

## Usage

Hit 'Deal', select or deselect cards that you want to hold. Hit 'Draw' and new cards will be drawn from the top of the deck.

Deck is always shuffled when you deal a new hand. 

Dollar amounts are added as you 'win' different hands.
